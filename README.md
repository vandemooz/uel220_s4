# Toobovan Project
**Course** : UE220 - advanced JavaScript | **Date:** October 2025

## About
An interactive quiz game that tests your knowledge about air pollution levels around the world. Players compare two cities and guess which one has higher PM2.5 pollution levels using real-time data from the OpenAQ API.
This project focuses on **API integration**, **asynchronous JavaScript**, and creating an **engaging user experience** with dynamic content.

## Features :
- Real-time air quality data from international cities
- Quiz with customizable number of questions
- Score tracking system
- Visual feedback with country flags
- Detailed pollution level explanations

## Technologies used :
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenAQ API
- Fetch API for asynchronous requests
- Responsive design
